# Python 简介

## 什么是 Python

Python 是一种高级编程语言，由 Guido van Rossum 于 1989 年首次发布。

## Python 的特点

- **简单易学**：语法简洁清晰，适合初学者
- **功能强大**：拥有丰富的标准库和第三方库
- **跨平台**：可以在 Windows、Mac 和 Linux 上运行
- **开源免费**：完全免费，可以自由分发

## Python 的应用领域

1. **Web 开发**：Django、Flask 等框架
2. **数据科学**：NumPy、Pandas、Matplotlib
3. **人工智能**：TensorFlow、PyTorch、scikit-learn
4. **自动化脚本**：系统管理和任务自动化

## 第一个 Python 程序

让我们写一个简单的 Python 程序：

```python
# 打印欢迎信息
print("Hello, World!")

# 简单的数学运算
a = 10
b = 20
print(f"a + b = {a + b}")
```

运行上面的代码，你将看到输出：
```
Hello, World!
a + b = 30
```

## Python 的版本

- Python 2.x（已于 2020 年停止维护）
- Python 3.x（当前主推版本，推荐使用）

你可以通过以下命令检查你的 Python 版本：

```bash
python --version
# 或
python3 --version
```

## 总结

Python 是一门强大而简单的编程语言，适合初学者入门编程。它广泛应用于 Web 开发、数据科学、人工智能等领域。
