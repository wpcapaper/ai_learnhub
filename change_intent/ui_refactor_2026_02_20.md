# UI重构设计意图文档

## 变更概述

本次对 AILearn Hub 用户端进行整体UI重构，采用"治愈系·奶油抹茶"莫兰迪色系配色方案，旨在打造一个亲和力强、柔和、舒适的学习系统界面。

## 设计理念

### 核心设计特征

1. **色彩：低饱和度莫兰迪色系**
   - 减少视觉疲劳，适合长时间学习
   - 去除刺眼的高对比度色块
   - 模仿现实生活中柔和光影下的物体质感

2. **光影：柔和弥漫阴影**
   - 阴影边缘模糊，范围较大
   - 不使用生硬的投影
   - 营造轻盈悬浮感

3. **圆角：大圆角设计**
   - 圆角半径 16px-24px
   - 圆润的形状产生心理上的"安全感"和"亲和力"
   - 降低用户的心理防御机制

4. **文字：深炭灰代替纯黑**
   - 避免纯黑在白底上产生的眩光
   - 深灰更柔和，阅读更舒适

## 配色方案

| 用途 | 颜色名 | 色值 | 说明 |
|------|--------|------|------|
| 背景色 | 暖灰白 | #FAFAF8 | 像宣纸一样的质感 |
| 主色 | 鼠尾草绿 | #8FA98F | 自然、生机、平静 |
| 辅助色 | 奶油黄 | #F2E8CF | 温暖、柔和 |
| 标题文字 | 深炭灰 | #333333 | 重要标题 |

| 用途 | 色值 | 说明 |
|------|------|------|

## 设计细节

### 圆角规范

- 小圆角（按钮、标签）: 12px
- 中圆角（输入框、小卡片）: 16px
- 大圆角（主卡片）: 20px
- 超大圆角（特殊容器）: 24px

### 阴影规范

```css
/* 小阴影：悬浮提示 */
--shadow-sm: 0 4px 12px rgba(0, 0, 0, 0.04);

/* 中阴影：卡片默认状态 */
--shadow-md: 0 8px 24px rgba(0, 0, 0, 0.06);

/* 大阴影：悬停状态 */
--shadow-lg: 0 12px 32px rgba(0, 0, 0, 0.08);
```

### 按钮样式

- 主按钮：鼠尾草绿背景 (#8FA98F)，白色文字
- 成功按钮：柔和绿背景 (#7FB77E)
- 错误/危险按钮：柔和红背景 (#CF8A8A)
- 大圆角 (16px)，柔和阴影

### 卡片样式

- 背景：纯白 (#FFFFFF)
- 圆角：20px
- 阴影：0 8px 24px rgba(0, 0, 0, 0.06)
- 悬停：阴影加深，轻微上浮效果

## 心理暗示

### 整体感受

这套配色方案传达的心理暗示：
- **自然**：鼠尾草绿让人联想到大自然、植物
- **生机**：低饱和度绿色给人生命力但不张扬
- **平静**：整体色调柔和，有助于集中注意力
- **温暖**：奶油黄点缀增添温馨感

### 与学习场景的契合

- 长时间阅读不累眼
- 答题时情绪平稳
- 正确/错误反馈不刺眼（柔和绿/红）
- 整体氛围舒适，减少学习焦虑

## 技术实现

### CSS变量系统

所有颜色、圆角、阴影均通过CSS变量定义，便于：
- 全局统一管理
- 快速调整配色
- 保持一致性

### 内联样式优先

考虑到Tailwind CSS的限制，采用内联`style`属性应用CSS变量，确保：
- 颜色精确控制
- 与Tailwind工具类共存
- 易于维护

## 预期效果

1. **视觉舒适度提升**：低饱和度配色减少眼部疲劳
2. **亲和力增强**：大圆角+柔和阴影营造友好氛围

4. **品牌一致性**：统一的设计语言贯穿整个系统
